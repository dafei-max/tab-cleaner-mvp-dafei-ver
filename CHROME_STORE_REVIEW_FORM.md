## 6. 远程代码使用说明

### 问题：您正在使用远程代码吗？

**选择：是的，我在使用远程代码**

### 理由（1000字以内）：

**填写内容（中文）：**

```
Tab Cleaner MVP 使用远程代码（后端 API）来实现以下核心功能，这些功能需要服务器端的处理能力，无法在浏览器扩展程序本地完成：

1. **OpenGraph 数据抓取**：从网页抓取元数据（标题、描述、缩略图）。虽然可以在扩展程序中直接访问网页，但为了统一处理逻辑、支持批量处理和错误处理，我们使用后端 API 来处理。

2. **AI 语义搜索**：调用 AI 模型（通义千问 qwen2.5-vl-embedding）生成文本和图像的向量表示，实现语义搜索功能。这需要调用阿里云 DashScope API，无法在浏览器本地完成。

3. **智能聚类**：使用 AI 技术（K-means 聚类算法和 AI 分类）自动将标签页按主题分类。这需要服务器端的计算资源。

4. **网页截图**：为文档类网页（如 GitHub、技术文档）生成截图，作为可视化内容。使用 Playwright 在服务器端生成截图，确保质量和一致性。

5. **文档卡片生成**：为没有 OpenGraph 图片的文档类网页生成 SVG 卡片，包含标题、类型和网站名称等信息。

技术实现：
- 后端 API 地址：https://tab-cleaner-mvp-production.up.railway.app（生产环境）
- 所有 API 调用都通过 HTTPS 加密传输
- 仅发送用户主动提交的标签页数据（URL、标题）
- 不收集或存储用户的个人信息或浏览历史

安全措施：
- 所有数据传输都经过加密（HTTPS）
- 后端 API 仅处理用户主动提交的数据
- 不会在后台自动访问或监控用户的浏览行为
- 所有远程代码调用都经过严格审查，确保安全性

数据使用：
- 标签页 URL 和元数据仅用于抓取 OpenGraph 数据和生成 AI 向量
- 搜索关键词仅用于生成查询向量和执行搜索
- 不会将数据用于广告或营销目的
- 不会与第三方共享数据
```
