Privacy Policy for Tab Cleaner MVP

Last Updated: December 3, 2025

1. Introduction

Tab Cleaner MVP ("we", "our", or "us") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and protect your information when you use our Chrome browser extension.

2. Information We Collect

Tab Cleaner MVP collects the following information:

  • Tab Information: When you click the "Clean" button, we collect the URLs and titles of your open browser tabs.
  • OpenGraph Metadata: We extract metadata (title, description, thumbnail images) from web pages to create visual cards for your tabs.
  • Image Data: When processing tab thumbnails, we analyze images to extract visual attributes including:
    • Color information (using Hex color codes for accurate color representation)
    • Image captions and descriptions
    • Design style tags (modern, minimalist, vintage, etc.)
    • Object identification (chairs, tables, plants, etc.)
    This analysis helps improve search accuracy and visual organization of your tabs.
  • Search Queries: If you use the AI search feature, we collect your search queries to generate semantic search results.
  • User Preferences: We store your preferences (canvas zoom level, search history, clustering configuration) locally on your device.
Important: We only collect information when you actively click the "Clean" button or use search features. We do not automatically collect or monitor your browsing behavior in the background.

3. How We Use Your Information

We use the collected information for the following purposes:

  • Tab Management: To organize and display your tabs on the visual canvas.
  • AI Processing: To provide semantic search and intelligent clustering features.
  • Image Analysis: To analyze images and extract visual attributes (colors, styles, objects) for improved search and organization. This includes:
    • Using AI vision models (Qwen-VL) to generate image captions and descriptions
    • Using color extraction algorithms (K-Means clustering) to identify dominant colors with Hex codes
    • Detecting subject regions in images to prioritize main object colors
    • Identifying design styles and objects for better categorization
  • OpenGraph Extraction: To extract metadata from web pages for visual card creation.
  • User Experience: To remember your preferences and improve the extension's functionality.

4. Data Storage

Local Storage: All tab metadata and user preferences are stored locally on your device using Chrome's chrome.storage.local API. This data is never automatically uploaded to remote servers.

Backend Processing: When you use features that require AI processing (semantic search, clustering), your tab URLs and metadata are temporarily sent to our backend server for processing. This data is:

  • Transmitted securely via HTTPS encryption
  • Processed only for the specific feature you requested
  • Not permanently stored on our servers
  • Not shared with third parties

5. Data Sharing

We do not share your data with third parties. We do not sell, rent, or trade your information. The only exception is when data is sent to our backend server for AI processing, which is necessary for the extension's core functionality.

Our backend server uses third-party AI services (Alibaba Cloud DashScope API) for semantic search, clustering, and image analysis. These services process your data according to their own privacy policies, but we ensure that:

  • Only necessary data (URLs, titles, metadata, and thumbnail images) is sent
  • All communication is encrypted via HTTPS
  • No personal information is included
  • Images are processed only to extract visual attributes (colors, styles, objects) and are not stored permanently
  • Color information is extracted using algorithms that identify dominant colors with Hex codes for accurate representation

6. User Control

You have full control over your data:

  • Data Collection: Data is only collected when you actively click the "Clean" button or use search features.
  • Data Deletion: You can clear all stored data at any time by clearing Chrome's extension storage or uninstalling the extension.
  • Opt-Out: You can stop using the extension at any time, which will prevent any further data collection.

7. Security

We implement the following security measures:

  • All network communication is encrypted via HTTPS
  • Data is stored locally on your device
  • Backend API uses secure authentication and encryption
  • No sensitive personal information is collected

8. Children's Privacy

Tab Cleaner MVP is not intended for children under the age of 13. We do not knowingly collect information from children under 13.

9. Changes to This Privacy Policy

We may update this Privacy Policy from time to time. We will notify you of any changes by updating the "Last Updated" date at the top of this page. We encourage you to review this Privacy Policy periodically.
